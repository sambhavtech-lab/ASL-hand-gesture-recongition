# ASL-hand-gesture-recongition
A Deep Learning based image classification project that recognizes American Sign Language (ASL) hand gestures using Convolutional Neural Networks (CNN).

📌 Project Overview
This project trains a CNN model to classify ASL alphabet hand gestures from images.
The model is trained using the ASL Alphabet dataset and evaluated on validation data.

📂 Dataset
Dataset Name: ASL Alphabet Dataset
Source: Kaggle
Link: https://www.kaggle.com/datasets/grassknoted/asl-alphabet

⚠️ Note: Dataset is not included in this repository due to large file size.
Please download it from Kaggle before running the project.
🛠 Technologies Used
Python
TensorFlow / Keras
NumPy
Matplotlib
Google Colab
🚀 How to Run the Project
Download dataset from Kaggle link above.
Extract the dataset.
Upload dataset folder to Google Colab.
Open the notebook.
Run all cells.

📊 Model Architecture
Convolutional Layers
MaxPooling Layers
Fully Connected Layers
Softmax Output Layer
